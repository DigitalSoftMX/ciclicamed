# static sendToServer()

Envía al servidor los resultados del estudio médico seleccionado. Mientras el estudio se sube al servidor, se actualiza el porcentaje de subida al asignar a la variable uploadPercentage el porcentaje actual de subido. Si la petición se procesa correctamente, se envía en evento afterSendData para indicar que finaliza la subida, se reinician todos los datos del formulario (tanto de archivos, como de formularios) y de muestra el componente AlertSuccessComponente con un mensaje de éxito. En caso contrario, se reinician todos los datos del formulario, se asigna a la variable errors los errores provistos por el servidor, se muestra el componente ErrorAlertComponent y se envía un evento afterSendData con los datos de error del servidor
