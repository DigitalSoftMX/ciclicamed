# static copyScheduleData(date)

Antes de mostrar el componente LateralScheduleComponent, verifica que el rol del usuario logueado actualmente no sea imagenologia o laboratorio. Una vez verificado el rol, obtiene el número del día seleccionado y busca si en la variable hoursEnabled existe un horario para el día seleccionado. Si el día seleccionado es igual o mayor al día actual y si existe un horario para el día seleccionado, entonces asigna a la variable scheduleSelected datos vacíos, asigna a la variable startHour y startMinute la hora y minuto (respectivamente) la hora y minuto de inicio de acuerdo al horario de trabajo encontrando anteriormente, a lo cual asigna a las variables scheduleSelected.consult_schedule_finish y scheduleSelected.consult_schedule_start el horario asignado en las variables startHour y startMinute. Finalmente abre el componente (@link LateralScheduleComponent)
Parameters:
Name Type Description
date string Fecha de la cita seleccionada
