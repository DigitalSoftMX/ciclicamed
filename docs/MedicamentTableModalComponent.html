# static createMedicament()

Envía el nuevo medicamento creado al servidor. En caso de que la petición se procese correctamente, se asigna al objeto successAlert un mensaje de éxito, a lo cual se procede a mostrar el componente SuccessAlertComponent. En caso contrario se asigna a la variable errors, los errores retornados por el servidor y se muestra el componente ErrorAlertComponent
