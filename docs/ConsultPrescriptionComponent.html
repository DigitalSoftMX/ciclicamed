# async static createPDF()

Crea un pdf con los datos de los medicamentos de la receta creada en la consulta tomando como base el archivo prescription.pdf. Para crear dicha receta primero convierte el pdf en un archivo legible de tipo ArrayBuffer para la librería PDF-LIB, despues procede a filtrar los medicamentos que sean creados por el doctor y se asignan a la variable filterPrescriptionList. Una vez realizado este paso, se procede a asignar los datos del doctor, paciente y receta al pdf Ver el proceso y se retorna el pdf creado
