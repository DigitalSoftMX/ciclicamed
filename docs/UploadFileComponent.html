# static uploadFile()

Envía al servidor los archivos seleccionados por el usuario. Mientras la petición es procesada se actualiza la variable uploadPercentage, para indicar el porcentaje de subida de los archivos. Si la petición se procesó correctamente, se muestral el modal SuccessAlertComponent, en caso contrario se asigna a la variable errors el mensaje de error provisto por el servidor y se muestra el componente ErrorAlertComponent. En ambos casos, la variable uploadPercentage se reinicia a 0 una vez finalizado la subida. Nota: solo aplica si la variable upload es true y solo se utiliza en la consulta médica
